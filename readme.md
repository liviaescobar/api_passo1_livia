criar pasta
```
mkdir NOME_PASTA
```
acessar pasta
```
cd NOME_PASTA
```
criar arquivos para documentar projeto
```
touch readme.md
```
iniciar o gerenciadro de pacotes Node
```
npm init -y
```
instalar os pacotes
```
npm i express nodemon dotenv
```
abrir o vscode
```
code . 
```
criararquivo .gitignore
```
nano .gitignore
```
Adicionar no arquivo .gitignore o nome da pasta criada após a instalação dos pacotes
* Ctrl + o: Salvar o arquivo
* Enter
* Ctrl + x: Fechar o arquivo
```
node_modules
```
Criar estrutura de arquivos e pastas
```
mkdir src
```
Criar arquivos dentro da pasta src
```
touch src/app.js
```
```
touch src/server.js
```
```
mkdir src/config
```
```
mkdir src/controllers
```
```
mkdir src/routes
```
Inicializar o gerenciador de arquivos
```
git init
```
Informar nome e email 
```
git config --global user.name "FIRST_NAME"
```
```
 git config --global user.email "EMAIL@EXAMPLE.COM"
```
verificar arquivos 
```
git status 
```
adicionar arquivos 
```
git add .
```
salvar e escrever comentário
``` 
git commit -m 'estrutura do projeto'
```
- criar repositório no GitHub
- clicar no icone de copiar o URL do repositório

definir a branch main
```
git branch -M main
```
colar a URL 
```
git remote add origin COLAR_URL
```
enviar para o GitHub
```
git push -u origin main
```
